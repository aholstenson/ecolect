Dates and times

Date

The date value captures the date of a single day and maps to a DateValue object.

Usage:

const { date } = require('ecolect/values');

// In a phrase
builder.value('idOfValue', date());

// As a matcher
const matcher = date().matcher(language);
await result = matcher('today');
